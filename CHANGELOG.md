# Changelog

All notable changes to the Wellness Tracker project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-28

### Added
- Initial release of Wellness Tracker
- User authentication (email/password and anonymous)
- Mood check-in feature with emoji selection
- Sleep and study balance tracker with graphs
- Meditation timer with customizable durations
- Anonymous peer support chat
- AI-powered journaling assistant
- Cross-platform support (iOS, Android, Windows)
- Firebase integration (Auth, Firestore, Realtime Database)
- Test-driven development structure
- Comprehensive documentation
- CI/CD pipeline with GitHub Actions

### Features

#### Authentication
- Email/password registration and login
- Anonymous sign-in for chat
- Password reset functionality
- Persistent authentication state

#### Mood Tracking
- 5-level mood scale (Very Happy to Very Sad)
- Optional notes for each mood entry
- Visual mood history

#### Sleep & Study Balance
- Sleep hours logging
- Study hours logging
- Visual graph showing balance over time
- Quality ratings for sleep
- Productivity ratings for study

#### Meditation
- Customizable timer (5, 10, 15, 20, 30 minutes)
- Pause/Resume functionality
- Session history tracking
- Completion notifications

#### Peer Support Chat
- Anonymous user identification
- Real-time messaging
- Group chat rooms
- Respectful communication guidelines

#### AI Journal
- Write journal entries
- Get AI-powered insights
- Entry history
- Tag system for organization

### Technical
- Clean architecture pattern
- Feature-first folder structure
- Provider for state management
- Comprehensive test coverage
- Firebase backend
- Material Design 3 UI
- Dark mode support
- Responsive design
